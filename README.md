## 古代偏旁通寫的規律

古代偏旁通寫最常見的一種，就是點的有無。如：王玉玊、土𡈽圡、丈𠀋、友犮叐等。

判斷一個疑難字是否存在通寫，將點補齊帶入元文通讀；因繁筆添贅，多出來的點，去點再帶入原文。能通，則此字為通寫，元字即爲訛寫。

其次，利用偏旁通寫，去推本來的漢字。再將通寫漢字帶入元文，能通，則元字為訛寫。

再次，利用上下文。釋讀某個人的筆跡，此人一定存在某種書寫習慣。如喜上下異構（峯鵞畧），不從左右異構字（峰鵝略）；或喜略字（尭黒曽边邉），不從正體（堯黑曾邉邊）；或喜繁筆（壁，下從圡；璧，下從玊；攵攴偏旁右上角加一點），不從簡筆。利用上下文可糾正曾經釋讀遇到的錯誤。

現代漢字中，兩個字意思及用途在寫本時代尚未分化，而現代漢語中已經發生明顯分化的漢字，（如粧裝妝、搃總縱、鬟嬛環环妚）判斷起來比較麻煩，不能按今天的思維去解。要按漢字六書規律去推測，是否引起了偏旁通寫。

有新的通寫内容可于本項目中 [更新 issue](https://github.com/Steve-Cheung-emct/old-written-kanji) ，我將持續更新此帖説明。

## コ几

此門通冩實為殳部漢字，以上部「几」訛作「コ」。易見段、叚混用，「段」常作「叚」形，實「段」字。

又，没、沒、役、設、股、投、歿、殷等，右側皆有此訛。
